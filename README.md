# LifeHUD-student-manager
# Title: LifeHUD - Personal Student Manager

LifeHUD is a simple Python program designed to help college students manage their daily finances and track their health habits using a command-line interface.

## Why i made this?:
Students often struggle with managing pocket money and maintaining a healthy lifestyle. This project helps solve that by providing a quick way to log expenses and calculate a daily "Health Score".

## Features :
* **Finance Manager:**:
  - Add expenses (money spent).
  - Add income (money received).
  - Automatically updates the current wallet balance.
* **Health Tracker:**
  - Calculates a score (0-100) based on sleep, diet, stress, and physical activity.
* **Data Persistence:**
  - Uses the `json` module to save all data to `HUD_data.json`.
  - Data is not lost when you close the program.
## Technologies Used:
  Python
## Python Modules Used:
* **json:** Used to store user data (Profile, Finance, Health) in a structured file.
* **os:** Used to check if the data file exists and to clear the screen for a clean UI.
* **time:** Used to create small delays for a better user experience.

## How to Run:
1. Ensure Python is installed on your system.
2. Download the `LifeHuD.py` file.
3. Open your terminal or command prompt in the folder.
4. Run the following command:
   ```bash
   python LifeHuD.py
